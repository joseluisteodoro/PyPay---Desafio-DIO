
# PyPay🌐

Olá! O PyPay🌐 faz parte do desafio para a DIO.
Se trata de uma versão simples de um sistema bancário feito em python .

Versão atual V2.0

## ✨ Funções disponíveis
- Saque  
- Depósito  
- Extrato
